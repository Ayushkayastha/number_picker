# number_picker

A customizable number picker widget for Flutter with infinite scroll, steps, and styled UI.

## Features
✅ Infinite looping picker  
✅ Step-based increments (e.g., 2, 5, 10)  
✅ Custom colors & text styles  
✅ Haptics support
